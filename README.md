# Border Surveillance AI System

This project implements an AI-based surveillance system designed for border security monitoring.

## Features
- Object Detection using YOLO
- Motion Detection
- Anomaly Detection
- Risk Level Prediction
- Alert Generation

## Technologies
- Python
- PyTorch
- YOLO (Ultralytics)
- OpenCV
- NumPy

## Project Structure
- data: surveillance dataset
- models: trained models
- modules: AI pipeline
- notebooks: EDA analysis
- alerts: generated alerts

## How to Run

pip install -r requirements.txt

python main.py
